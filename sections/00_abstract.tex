\section*{Abstract}
\label{sec:abstract}
This project research studies how machine learning applications are applied inside of macroeconomic field. Using two different models, LSTM for gdp direction predictions and XGBoost for classifying the state of the economy (recession, slowdown, recovery, expansion), research dives into which purposes each model is beneficial. Results of the research is efficient model for classifying model economy with a high accuracy.
